The beginning of the Operating System

Valeo24 proposed the idea of collaboration between ScreamingQQ originally. ScreamingQQ agreed to take on the job and then they started the large and taxing project. Currently right now the OS is being developed. Version 1.0 alpha is currently out in the GalacticTeamStudio.

Is Galactic OS affiliated with OS Wars?

Sadly we are not affiliated with OS Wars. We might join in the battle OS Wars in the future but we were too late to start in OS Wars X. If we have any supporters for us entering in OS Wars XI, then we will consider the opportunity, but for now we are currently building up the ultimate team.

What is Galactic OS?

Galactic OS is a software on Scratch created by the GalacticTeam. This OS has capabilities that no other OS on Scratch has ever had before. Some of these features include a VPN, a security system, a webbrowser, netflix, and even more. If you want to see more features coming to Galactic OS, please support us by Following our admins and also like and favorite this project on scratch.
